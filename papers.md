# Papers and Preprints

The written work is published as preprints and revised iteratively. Later papers build on earlier ones, and changes reflect feedback, testing, and clarification over time.

## Recursive Division Tree
**Recursive Division Tree: A Log-Log Algorithm for Integer Depth**  
https://doi.org/10.5281/zenodo.17487651  

Introduces a deterministic recursive decomposition on the integers that induces depth, ancestry, and a partial order. Establishes saturation behavior of recursive depth.

## Recursive-Adic Numbers
**The Recursive-Adic Number Field: Construction, Analysis, and Recursive Depth Transforms**  
https://doi.org/10.5281/zenodo.17555644  

Develops a valuation system based on recursive depth rather than prime divisibility. Defines incidence algebra, zeta and Möbius transforms, and non-Archimedean-style behavior derived from algorithmic hierarchy.

## Recursive Entropy
**Recursive Geometric Entropy: A Unified Framework for Information-Theoretic Shape Analysis**  
https://doi.org/10.5281/zenodo.17882310  

Defines entropy-like quantities that evolve across recursive refinement. Establishes bounds and limiting behavior for hierarchical entropy growth.

## Related Applied Work
**A Unified Closure Framework for Euler Potentials in Resistive MHD**  
https://doi.org/10.5281/zenodo.17989242  

An applied project that explores closure constraints in magnetohydrodynamics. Included here because it informed later work on structure, constraints, and stability.

### Notes on Revision
Some papers exist in multiple versions. Changes reflect external feedback, testing results, and improvements in clarity and framing. The progression is intentional and documented.
